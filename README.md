# Shopping Cart

A responsive shopping cart web application built with HTML, CSS, JavaScript, and Bootstrap. The project simulates a simple e-commerce experience with client-side authentication and cart management using Local Storage.

## Features

- User Sign Up
- User Sign In
- Product Listing
- Add Items to Cart
- Remove Items from Cart
- Update Cart Quantity
- Cart Total Calculation
- Persistent Data using Local Storage
- Responsive Design

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Local Storage

## Purpose

This project was built to practice front-end web development fundamentals, including DOM manipulation, responsive design, user authentication, and client-side data persistence.
